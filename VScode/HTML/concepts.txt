TCP/IP : Used to get data from A to B

IP = Internet Protocol - Is a protocol that computers speak and allows them to know how to adress computers in the internet
Each computer have a unique adress(IP) #.#.#.# 32 bits --version 4  (--version 6 = 128 bits)
                                       ^ ^ ^ ^       
                                       8 8 8 8 bits each which varies from 0 to 255

TCP = Transmition Control Protocol - It's a different set conventions that computers adhere to in order to solve some particulars problems
Used for distinguishing one type of service from another.


80 - HTTP : Hyper Text Transfer Protocol

443 - HTTPS : "S" means secure for privacy sake
